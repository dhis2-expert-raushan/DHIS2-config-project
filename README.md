# DHIS2-config-project
DHIS2 metadata modeling, configuration and KPI dashboard for health data
## Overview
This project demonstrates my experience in configuring DHIS2 metadata and building dashboards for health data analysis.

## Tools Used
- DHIS2
- Power BI
- Excel

## Key Work
- Identifying system content requirements, data flow
- creating metadata model
- Metadata configuration (data elements, category options, data sets, programs(event/tracker), indicators)
- Validation rules setup
- Dashboard creation withing DHIS2 or outside DHIS2 such as Power BI, Tableau etc. for visualizing health KPIs.
  ## Output
- Configure and implement DHIS2/HMIS system based on data requirments of an orgnization.
- Interactive DHIS2 or Power BI dashboard
- Cleaned and structured Indicators outcome

## About Me
I have 10+ years of experience working with health information systems including DHIS2 and HMIS.
